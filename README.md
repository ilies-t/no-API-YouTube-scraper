## How it work  
#### Get the basic meta of a YouTube channel without any API, only scraping.  
* Channel name  
* Banner  
* Profile image  
* Subscribers  
    
## Built With  
* [PHP Simple HTML DOM Parser](http://sourceforge.net/projects/simplehtmldom/) - To parse HTML  
  
Made with PHP, Sass, jQuery
